# Artificial-Intelligence-in-Clinical-Trial-Participant-Recruitment-and-Retention

## Overview
This repository contains the analysis code and supporting data for the manuscript

The study systematically reviews and quantitatively synthesizes evidence on the use of AI-driven tools across clinical trial recruitment and retention workflows, including eligibility assessment, patient screening, cohort identification, trial matching, and retention support.

## Methods Summary
- Review framework: Joanna Briggs Institute (JBI)
- Reporting standard: PRISMA-ScR
- Databases searched: MEDLINE, Embase, CINAHL, Cochrane Library, Scopus
- Time window: January 2018 – June 2024
- Meta-analysis: Random-effects models stratified by recruitment/retention task
- Performance metrics: sensitivity, specificity, precision, F1-score, AUC

Large language model–assisted screening (ChatGPT, OpenAI) was used to prioritize title and abstract review, with **all records manually verified by human reviewers**.
